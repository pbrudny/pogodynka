## Setup
Install packages

`yarn install`

Set env variables. 

You can use sample: 

`cp .env.sample .env` 
## Run

In the project directory run:

 `yarn start`

Runs the app in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.


## Tests
Launching Cypress to run Cucumber tests
- Start app `yarn start`
- Open cypress `yarn run cypress open`
- Click 'Run all specs'
- Watch scenarios being tested on the Electron browser
